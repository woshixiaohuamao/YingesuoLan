输出一个json格式数据
包含3个字段：
prediction_start：预测开始时间
prediction_interval：预测时间间隔
predictions：预测结果列表，每个元素包含2个字段：
 - timestamp：预测时间戳
 - value：预测值

预测长度：60分钟。共预测从调用时间开始往后60个点的总用气量

响应示例：
{
	"prediction_start": "2025-05-27T14:12:27.700467",
	"prediction_interval": "PT1M",
	"predictions": [
		{
			"timestamp": "2025-05-27T14:13:27.700467",
			"value": 219.64700317382812,
			"upper_bound": 286.43259195950895,
			"lower_bound": 152.86141438814727
		},]
}
